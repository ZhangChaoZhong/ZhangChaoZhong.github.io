项目经历

在线人脸考勤小程序开发 2022.03~2022.06

简介：该项目采用前后端分离开发模式，前端使用uni-app，后端使用SpringBoot，实现注册登录，人脸考勤签到，系统通知功能。

  1. 为加强系统的安全性，自定义xssFliter 对请求数据进行转义，抵御跨站脚本XSS攻击
  2. 利用Spring AOP+Shiro+JWT实现用户鉴权和RBAC模型的资源控制
  3. 利用ThreadLocal+Redis 实现令牌的自动刷新，降低令牌泄露的风险以及提高用户体验
  4. 利用人脸识别和地理位置实现复杂业务的人脸考勤签到功能
  5. 利用RabbitMQ的队列机制进行削峰，缓解高并发情况下的系统资源短缺问题
  6. 使用线程池异步发送消息，提高系统响应速度同时降低资源消耗

校内经历

莞工教学助手小程序开发 2019.03~2020.06

简介：独立开发的基于CS架构的微信小程序，集成多个API 接口，为师生提供便捷的查询服务。

  1. 先通过检测服务器编码格式，再统一编码的方法解决中文乱码问题
  2. 利用爬虫技术模拟登录突破登录限制，自动爬取教务处上的信息，例如：成绩，课表等
  3. 使用 Jsoup解析页面数据，存入数据库中并集成 JSON API，为小程序提供数据来源
  4. 使用Ngrok 实现内网渗透，访问内网的 FTP服务器，解决外网无法访问内网的难题
  5. 利用动态 IP代理技术，解决因频繁访问校园服务器导致腾讯云服务器 IP被封的问题
  6. 将项目部署到腾讯云服务器，并在微信小程序上发布上线，该项目获得中国大学生计算机设计大赛国赛二等奖
