莞工教学助手

发表于 2022-03-02 更新于 2022-07-02 分类于 项目作品 Valine：

展示

背景

解决方案

难点

功能

（一）学生模块

1.1 学生用户登录

输入中央认证的账号密码登录，登录后才可以查询成绩，课表等。

1.2 学生用户点击微信授权登录

微信登录才可以发布话题

1.3 点击发布话题

点击后就可以进入发布话题界面，就可以发布话题了

1.4 点击发送话题

点击后，就可以跳转到话题列表页面

1.5 点击点赞

1.6 点击话题

进入话题详情页面

1.7 点击发送评论

评论区就会出现评论的内容

1.8 点击评论内容进行回复

评论区就会出现回复的内容

1.9 用音频进行评论

评论区就会出现音频的内容

1.10 点击作业

查看作业列表

2.11 点击学生服务中心的成绩

查看学业成绩，以此类推可以查询其他服务

（二）倒计时小工具

2.1 点击添加倒计时

输入倒数日主题，时间，就可以生成倒数日了

2.2 点击某倒计数日

进入倒数日详情，可以进行修改

（三）教师模块

3.1 教师用户登录

输入中央认证的账号密码登录，登录后才可以查询成绩，课表等。

3.2 教师点击用户微信授权登录

微信登录才可以发布作业

3.3 教师点击课表

查看课表信息

3.4 教师点击数字图像处理的学生名单时

查看学生名单

3.5 教师用户点击数字图像处理实验二的作业提交情况时

查看 ftp 上作业的提交情况
